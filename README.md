# gesture

鼠标及触控手势事件库



## 项目结构



1. pointer 目录

   增加单独控制阈值的手势 matrix 计算

   统一封装事件监听

2. test 事件测试



## 方案

> 给我一个手势库的方案，包括鼠标、手势的时间输入，在手势中支持多点。需要包含通用的手势场景，如单击、双击、拖动、滑动、双指缩放等

包含鼠标、手势的事件库

1. 事件监听
   1. 统一事件源，抽象接口整合 mouse、touch
   2. 手势事件的 touch 的多点支持
2. 手势识别算法
   1. 单击
   2. 双击
   3. 单指拖动、滑动
   4. 双指缩放、移动
   5. 识别手指、手掌
3. 事件传播机制
   1. 各内部事件触发与不触发控制机制
4. 触发点数据，触发事件每个点数据都需要
   1. 唯一标识
   2. 触发目标 dom
   3. 坐标
   4. 时间戳
5. 性能
   1. 节流：指定间隔时间内只触发一次
